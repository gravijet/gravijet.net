What is gravijet.net?

On gravijet.net you can practice your Minecraft skills. We have several gamemodes for you to competete against other players. Fight and practice your Minecraft skills!

User friendly

Our team is very user friendly and positive. Don't mind to ask someone if you need help. 

Fast support

If you need support, you can simply create a ticket in our Discord and we will help you as fast as we can. 

How to join

Here you can see, how you can join on the Minecraft server.

1

Start Minecraft

Start any Minecraft Version and click on Multiplayer.
2

Add the server

Add a Server and enter gravijet.net in the server address field and click "Done".
3

You're done!

Now you can play on gravijet.net with other players!
